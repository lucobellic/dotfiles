# Instructions

- Always use Context7 MCP when you need library/API documentation,
  code generation, setup or configuration steps.

- For general questions, or when you require more knowledge or information,
  perform a web search to gather information online.

- Use ccc MCP for semantic search.

- Prefer usage of `rg` and `fd` instead of `grep` and `find` for searching through files.

- Do not call subagent.
